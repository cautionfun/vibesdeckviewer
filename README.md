# vibesdeckviewer
A site for viewing and sharing Vibes decklists.
Also features a deck building page for building/editing/important/exporting Vibes deck lists. It also allows you to search for a card's text, along with the typical filtering/search functionality.

Main page: https://cautionfun.github.io/vibesdeckviewer/index.html

Search page: https://cautionfun.github.io/vibesdeckviewer/deckbuilder.html

Share links to a decklist by appending ?deck={deck_code_copied_from_client} to the page's URL.
